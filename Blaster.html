Privilege Escalation

Through Command Prompt it is visible that CVE-2019-1388 is the vulnerability that the privileged user searched for on the system.

This is the file that the privileged user deleted but forgot to permanently delete. This file requires administrator privileges to run the application.

But through this vulnerability we can leverage the invoke of administrative commands. If we click the link to the publisher’s certificate it does not open but it is in the history of the internet explorer.

Open the website and save it with filename as *.*, this is the error the dialog box will not close after saving.

Now search for cmd application on the system32 folder and open it with normal privs.

It will provide administrator privileges i.e. NT Authority System.
